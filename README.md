# Investor Trend Analysis Dashboard using Power BI

This project analyzes investor behavior, investment patterns, startup performance, ROI trends, and investor engagement using Power BI. The dashboard helps businesses understand investment distribution, cash flow, profitable sectors, funding-stage performance, and startup growth potential.

## Objective

To build an interactive Power BI dashboard that provides business insights on investor trends, investment returns, startup valuation, follow-on investments, and board seat participation.

## Tools Used

* Power BI
* Excel
* DAX
* Data Cleaning & Preprocessing
* Data Visualization

## Key KPIs

* Total Investors
* Total Investment Amount
* Average Investment Amount
* Average ROI
* Success Rate
* Follow-On Investment Rate
* Board Seat Participation Rate
* Average Holding Period
* Investor Satisfaction Score

## Dashboard Pages

1. **Executive Summary**
   Shows overall KPIs, investment by industry, ROI by industry, investor type distribution, and funding stage analysis.

2. **Investor and Startup Analysis**
   Analyzes investor types, top startups by investment, top startups by ROI, board seat participation, and follow-on investment trends.

3. **Geographic and Data Quality Analysis**
   Shows investment amount by country, ROI by country, investors by country, and data quality summary.

## Key Findings

* Total investment amount reached **$1.85 Billion**.
* Average ROI was **17.46%**.
* Investment success rate was **94.9%**.
* FinTech attracted the highest investment volume.
* SaaS generated the highest average ROI.
* Angel investors contributed the largest capital.
* Canada received the highest investment volume.
* Follow-on investment rate was **48.57%**.
* Board seat participation rate was **48.12%**.

## Project Outcome

The dashboard provides clear and interactive insights for understanding investor behavior, startup performance, investment profitability, and global investment distribution.
